🛍️ AI Shopping Assistant
An AI-powered shopping assistant built using React and modern frontend architecture that helps users discover products
intelligently through natural language recommendations, budget filtering, wishlist/cart management, and admin product controls.

✨ Features----------------------
🤖 AI Product Recommendation
Users can enter queries like:

I need a laptop under ₹60,000 for coding and college

The application provides:----------------------
🧠 Smart product suggestions
💰 Budget-based filtering
📊 AI-style explanation
⭐ Best choice recommendation

📦 Product Management----------------------

🖥️ Product Listing:
🎴 Dynamic product cards
📝 Product information display
⭐ Rating and stock visibility
📱 Responsive UI

🛒 Product Features:
❤️ Wishlist functionality
🛍️ Cart management
✨ Hover animations
🌙 Modern corporate UI

Admin panel supports----------------------
➕ Add product
❌ Delete product
📦 Manage stock
🔄 Product updates

Frontend authentication system includes----------------------
🔑 Login / Signup UI
👤 Role-based logic
🛡️ JWT-ready architecture
👨‍💻 Admin role simulation
