# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/tanyacodes-gif/ai-shopping-assistant.git
cd ai-shopping-assistant
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

# 🔐 Environment Variables

Create a `.env` file inside `/backend`

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
PORT=5000
```
